The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- 维度感知安全传送系统：
  - 全屏大地图右键传送引入维度感知安全管线，按维度执行不同的安全策略。
  - 地表传送：避免传送到岩浆、水、火等危险方块，通过螺旋搜索附近安全落脚点。
  - 洞穴传送：以玩家当前 Y 为参考上下扫描可安全站立位置（air+air+solid），限制搜索范围在玩家 Y±48 内，避免穿过天花板传送到地表。
  - 下界传送：强制两阶段探测流程——先传送到 Y=128（基岩顶上方），等区块加载后从 Y=125 向下搜索安全落脚点再进行第二次传送；搜索范围限制在 [2, 125]，避免传送到顶部/底部基岩层；若无安全落脚点则拦截传送。
  - 末地传送：不使用裸 Y 兜底，探测失败时回退玩家原位，避免掉入虚空。
  - 两阶段探测机制：Phase 0 传送到安全高度触发区块加载，Phase 1 等待区块就绪后进行稳定性检查（连续3帧返回相同Y）并执行第二次传送。
  - 二次验证：第二次传送前重新读取目标位置方块，确认 air+air+solid 条件，验证失败则拦截传送。

- 洞穴地图功能：
  - 精确复刻洞穴地图实现，包含地表方块过滤、地下检测、深度亮度衰减、列扫描等核心算法。
  - 洞穴模式选项简化为"关闭"和"开启"两种，移除了已废弃的"完整"模式。
  - 自动检测逻辑在玩家头顶有遮挡时激活洞穴模式，并跳过树叶、花草、草丛、藤蔓等覆盖方块，避免在树下误激活。
  - 亮度衰减公式区分可读模式与不可读模式，岩浆和水保持原色不衰减。
  - 全屏大地图左下角添加洞穴模式控制面板，包含模式下拉菜单和 Top Y 滑块（范围 -64~319）。
  - 洞穴设置独立面板，包含深度输入框、微调按钮和重置功能。

- 下界与末地地图支持：
  - 下界五种生物群系（下界荒地、绯红森林、诡异森林、灵魂沙峡谷、玄武岩三角洲）着色。
  - 下界专用地表 Y 检测函数，从 Y=123 向下扫描非空气方块。
  - 下界扫描参数：起始 Y = min(120, 玩家Y+40)，扫描深度 80，深度亮度衰减。
  - 末地生物群系识别与显示。
  - 移除下界全屏大地图的维度检查限制，使其可正常渲染。

- 下界与末地地图持久化与扩展：
  - 下界地图数据按维度独立存储，缓存目录隔离。
  - 维度切换时保存当前维度脏数据并清空缓存，防止跨维度数据混合。
  - 异步写入线程添加跨维度写入保护，维度变化时丢弃过时数据。
  - 游戏关闭时保存所有脏数据，确保退出时不丢失。
  - 统一渲染路径，下界/末地全屏大地图使用缓存区域纹理渲染，加载所有已保存数据，实现和主世界一样的地图不断扩大效果。

- 生物群系与颜色系统：
  - 生物群系名称翻译从 if-else 链重构为数据驱动的有序子串规则表（96 条规则），兼容基岩版旧 ID 与 Java 版新 ID。
  - 生物群系翻译字典扩展至 62 个键×16 种语言，修正黑森林、风袭丘陵等错误翻译。
  - 地表全屏大地图鼠标悬停显示对应位置生物群系名称，仅在区块已加载时显示，避免缓存数据错误。
  - 生物群系色调系统校准：黑森林树叶、苍白之园草地/树叶、森林树叶等基于截图采样精确匹配实际显示颜色。
  - 方块颜色修复：钟乳石方块和滴水石锥统一为精确色值 (0.525, 0.424, 0.365)；修复方解石、黑曜石、紫珀块、深板岩等 12 个方块颜色错误；灵魂沙规则前移避免被沙子规则误匹配。
  - 维度切换时立即中止当前地图扫描，防止跨维度数据混合。

- UI 与交互改进：
  - 小地图添加视野范围设置（10~200），控制可见方块数量。
  - 小地图布局设置每项添加独立重置按钮（↺ 图标）。
  - 快捷键设置面板增强：单行重置/清除/撤销功能，Ctrl+Z 撤销快捷键，每项独立重置到默认值和清除设置功能。
  - 路径点管理器添加多选功能：复选框、全选/清空/批量删除按钮。
  - 小地图路径点显示开关，支持状态持久化和实时切换。
  - 洞穴设置输入框宽度增大以容纳两位数，窗口长度调整避免垂直滚动。

- LeviLamina API 现代化：
  - 将自研的 LanguageManager 底层翻译解析重构为 LeviLamina 的 ll::i18n 架构，实现本地静态线程安全缓存防范悬空指针。
  - MapCacheManager 的 IOWorker 协程化，使用 ll::coro::CoroTask 和 keepThis().launch()。
  - 类型安全的版本比较，使用 ll::data::Version 替代原始字符串比较。
  - 统一使用 Mod 路径访问器（getLangDir()、getDataDir()、getConfigDir()），带旧路径迁移。
  - 传送机制从剪贴板+聊天栏改为直接 CommandRequestPacket 调用，避免位置回滚。

### Changed

- 优化小地图操作交互：在小地图隐藏状态下屏蔽 Y 键切换地图形状功能，避免误触。
- 洞穴模式检测冷却从 10 帧降至 5 帧，扫描时间预算从 2ms/帧提升至 4ms/帧，移动触发阈值从 16 格降至 8 格，静止刷新间隔从 5 秒降至 2.5 秒。
- 移除小地图下方的"洞穴地图已激活"标签。
- 移除洞穴地图快捷键（H 键），洞穴模式仅通过设置面板切换。

### Fixed

- 修正玩家坐标显示高度偏差：将位置接口从 player->getPosition() 修正为 player->getFeetPos()，消除 +1.62 轴高度偏移。
- 修复传送位置回滚：客户端 Actor::teleport 仅修改本地坐标被服务器覆盖，改用 CommandRequestPacket 发送 /tp 命令。
- 修复传送至未加载区域的概率性异常：两阶段探测机制 + 三轮螺旋搜索 + 验证后兜底。
- 修复下界传送卡在方块里：强制两阶段探测 + Y=128 安全高度 + 从 Y=125 向下搜索 + 二次验证。
- 修复下界传送到顶层基岩之上：搜索范围限制在 [2, 125]。
- 修复洞穴传送传送到地表：搜索范围限制在玩家 Y±48 内。
- 修复末地传送掉入虚空：不使用裸 Y 兜底，探测失败回退原位。
- 修复地表地图灰色石头色块：SafeGetSurfaceY 在部分加载区块返回洞穴顶板 Y 值，添加 IsStoneLikeBlock 检查和缓存地表 Y 回退。
- 修复退出游戏崩溃：disable() 函数未正确释放 DX11 钩子和 D3D/ImGui 资源，添加 g_isShuttingDown 标志和 shutdown() 函数。
- 修复部分加载区块访问崩溃：SafeGetSurfaceY 的 __try/__except 无法捕获 CFG 安全异常，改用 try/catch(...) 配合 /EHa。
- 修复 ImGui ID 冲突警告：renderRow lambda 被多行复用导致按钮标签相同，添加 PushID/PopID 和 ##后缀。
- 修复下界地图持久化失效：洞穴/下界扫描完成未调用 UpdateFromScan，添加异步写入线程。
- 修复下界全屏大地图不显示已保存数据：洞穴/下界模式使用独立渲染路径只渲染实时纹理，统一为缓存区域纹理渲染。
- 修复钟乳石方块和滴水石锥颜色与实际不符：统一为精确色值 (0.525, 0.424, 0.365)。
- 修复灵魂沙显示为沙子颜色：soul_sand 规则前移至 sand 规则之前。
- 修复黑森林被误识别为普通森林：roofed_forest 规则前移至 forest 规则之前。
- 修复苍白花园颜色过亮：基于截图采样匹配实际去饱和灰绿色。
- 修复森林树叶颜色过暗：基于截图采样匹配实际树叶颜色。
